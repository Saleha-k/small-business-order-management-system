# Small Business Order Management & Tracking System

## Overview

The Small Business Order Management & Tracking System is a software engineering project designed to help small businesses manage customer orders digitally instead of relying on manual methods such as WhatsApp messages, phone calls, and handwritten records.

The project focuses on requirement engineering, system analysis, software design, UML modeling, and UI prototyping.

---

## Problem Statement

Small businesses often struggle with:

- Manual order tracking
- Lost customer records
- Duplicate orders
- Lack of sales reporting
- Poor order status visibility

This system proposes a centralized platform for managing and tracking customer orders efficiently.
